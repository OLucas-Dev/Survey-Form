# Survey Form

This project is a survey form created using HTML as part of the
freeCodeCamp Responsive Web Design Certification.

## Features
- Form validation
- Input fields (text, email, number)
- Radio buttons and checkboxes
- Dropdown menus
- Textarea for comments

## Technologies
- HTML5
